Добавлен сервис, реализующий следующий функционал:
- поиск по переданному в запросе изображению наиболее подходящих из базы данных
  изображений путем поиска наибольших косинусных мер между эмбеддингами 
  изображений, вычисленными моделью ResNet34;
- поиск по переданному в запросе тексту наиболее подходящих изображений из базы 
  данных путем поиска наибольших косинусных мер между эмбеддингами изображений и
  текста, вычисленными моделью CLIP.

Для осуществления возможности сравнения векторных величин было добавлено 
расширение pgvector и добавлена hnsw-индексация методом косинусных мер на два
столбца эмбеддингов.

Внесены следующие изменения в уже реализованные ранее сервисы:
- добавлены новые роуты в сервис back для работы с новым сервисом;
- добавлены новые роуты, а также методы в класс базы данных в сервисе db_service
  для обработки запросов получения и изменения записей с эмбеддингами.
